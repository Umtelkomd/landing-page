Instrucciones para generar favicons

Este archivo contiene instrucciones para generar los archivos de favicon faltantes para el sitio glassfaser-utk.de.

Paso 1: Usar el favicon.svg como base

Ya hemos creado un archivo SVG optimizado (/public/images/favicons/favicon.svg) que puede utilizarse como punto de partida.

Paso 2: Generar los archivos PNG

Necesitarás generar los siguientes archivos:

  • favicon-16x16.png (16×16 píxeles)
  • favicon-32x32.png (32×32 píxeles)
  • apple-touch-icon.png (180×180 píxeles)
  • android-chrome-192x192.png (192×192 píxeles)
  • android-chrome-512x512.png (512×512 píxeles)

Puedes generar estos archivos de varias formas:

  1. Herramienta online: Visita Real Favicon Generator y sube el archivo favicon.svg.
  2. Software gráfico: Abre el SVG en Adobe Illustrator, Inkscape o similar y exporta en los tamaños requeridos.
  3. Línea de comandos: Usa herramientas como ImageMagick o librsvg para convertir SVG a PNG en diferentes tamaños.

Paso 3: Crear favicon.ico

También necesitarás un archivo favicon.ico que incluya múltiples tamaños (16x16, 32x32):

  1. Puedes usar favicon-generator.org para convertir PNG a ICO
  2. O usar la opción de Real Favicon Generator que crea este archivo automáticamente

Paso 4: Ubicar los archivos

Coloca todos los archivos generados en la carpeta /public/images/favicons/

Nota: Una vez que hayas generado todos los archivos y los hayas colocado en la ubicación correcta, puedes eliminar este archivo HTML de instrucciones antes de construir la versión final para producción.
